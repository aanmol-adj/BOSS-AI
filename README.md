# 🤖 BOSS AI

BOSS AI is a futuristic premium AI assistant developed using Python, Flask, HTML, CSS, JavaScript and OpenRouter AI integration.

---

# 🚀 Features

- Advanced AI Chatbot
- Hindi & English Support
- Real-Time AI Responses
- Futuristic UI Design
- Animated Interface
- Chat History
- Responsive Layout
- Smart AI Interaction
- Professional Dashboard
- Modern Glassmorphism Design

---

# 🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- OpenRouter AI API
